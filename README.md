# Morse Code App
This is an app that helps you talk via morse code using you android phone.

This app will contain a lot of features.
1. Broadcast morse via flashlight
2. Broadcast morse via speakers
3. Press '-' or '.' to show the letters
4. Type in english words (app translates)
5. Morse code cheat-sheet tab

The interesting thing about this project is that it is being 
developed only using one linux terminal.
